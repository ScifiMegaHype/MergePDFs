# PDF Merger

A one step process that easily merges PDFs; **no paywall**.  
Built with **python**.

---

# To EXE
To convert the python file to an exe, run `python -m PyInstaller --onefile -w --icon="adobe.ico" --add-data "adobe.ico;." merge_pdfs.py`

# Usage
1. Run the merge_pdfs.py python code
         **OR**
2. Run the merge_pdfs.exe
        **THEN**

  - Add, Combine and Enjoy.
